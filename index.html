// 01 · SOBRE EL PROYECTO

¿Qué es GlassIA?

GlassIA es un prototipo de gafas inteligentes que integra visión artificial, una micro-pantalla OLED y conectividad Wi-Fi. El dispositivo captura imágenes del entorno mediante un módulo de cámara, identifica componentes de hardware de computadoras y envía la información detallada a una página web para su visualización en tiempo real.

Nace de la curiosidad por entender cómo funcionan las cosas. No son solo unas gafas; es una compañera de aprendizaje que utiliza inteligencia artificial para que el hardware deje de ser un misterio y se convierta en algo fácil de comprender para cualquier estudiante.

A través del reconocimiento visual en tiempo real, permite identificar componentes y acceder a información de manera inmediata, fomentando un aprendizaje más interactivo, práctico y accesible dentro de entornos educativos tecnológicos.

// 02 · FUNCIONAMIENTO

¿Cómo Funciona?

01
📷

CAPTURA DE IMAGEN

El módulo de cámara integrado en las gafas captura imágenes del componente de hardware en tiempo real desde el punto de vista del usuario.

02
🧠

PROCESAMIENTO IA

El modelo de inteligencia artificial analiza la imagen y clasifica el componente detectado entre CPU, RAM, GPU, placa madre, PSU, fan, SSD o HDD.

03
📡

TRANSMISIÓN Wi-Fi

Los datos del componente identificado se transmiten de forma inalámbrica a través de Wi-Fi hacia la plataforma web.

04
🖥️

VISUALIZACIÓN OLED

La micro-pantalla OLED integrada en las gafas muestra el resultado del reconocimiento directamente al usuario sin interrumpir su visión.

05
🌐

PLATAFORMA WEB

La página web recibe los datos y despliega información detallada del componente: nombre, función, especificaciones y recursos de aprendizaje.
